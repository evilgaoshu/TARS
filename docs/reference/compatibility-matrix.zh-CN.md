# 兼容性矩阵

本文档定义了与 TARS 集成的第三方系统经过验证的版本和支持级别。

## 1. 指标连接器

| 系统 | 支持的版本 | 状态 | 备注 |
|--------|-------------------|--------|-------|
| VictoriaMetrics | v1.x | 经验证 | 推荐用于高性能指标。 |
| Prometheus | v2.x | 支持 | 完全兼容 Prometheus API。 |

## 2. 执行连接器

| 系统 | 支持的版本 | 状态 | 备注 |
|--------|-------------------|--------|-------|
| JumpServer | v3.x, v4.x | 经验证 | 支持 Job 执行 API。 |
| OpenSSH | 任何现代版本 | 支持 | 通过密钥或密码直接进行 SSH 执行。 |

## 3. 消息渠道

| 系统 | 能力 | 状态 | 备注 |
|--------|--------------|--------|-------|
| Telegram | 完整 | 经验证 | 直接机器人通信和内联按钮。 |
| Web 聊天 | 有限 | 实验性 | 内部控制台消息传递。 |

## 4. 身份提供者

| 系统 | 协议 | 状态 | 备注 |
|--------|----------|--------|-------|
| Google | OIDC | 经验证 | |
| GitHub | OAuth2 | 支持 | |
| 通用 OIDC | OIDC | 支持 | 任何符合标准的提供者。 |

## 5. 数据库要求

| 系统 | 最低版本 | 状态 | 备注 |
|--------|-------------|--------|-------|
| PostgreSQL | 15+ | 经验证 | 主工作流存储。 |
| SQLite | 3.x | 经验证 | 用于本地向量索引 (sqlite-vec)。 |
